# Roadmap

## Phase 1

- bootstrap Tauri + React + TypeScript
- initialize SQLite schema
- support roots and first scan
- expose search and status commands
- build launcher-style UI shell

## Phase 2

- add exclude rules
- add debounced watcher pipeline
- support rebuild and periodic reconciliation scans
- improve ranking and recency boosts

## Phase 3

- add tray support
- add start-on-login
- integrate global shortcut flow for Linux desktops
- improve desktop packaging

## Phase 4

- split indexing into a background daemon if needed
- handle larger indexes and performance profiling
- add extension filters and advanced search affordances
